配置Auto-GPT

🚀 特征

  • 🌐 用于搜索和信息收集的互联网接入

  • 💾 长期和短期记忆管理

  • 🧠 用于文本生成的 GPT-4 实例

  • 🔗 访问热门网站和平台

  • 🗃️ 使用 GPT-3.5 进行文件存储和摘要

📋 环境要求

  • 环境（选择其中之一）

    • VSCode + devcontainer: 已在 .devcontainer 文件夹中配置，可直接使用

    • Docker

    • Python 3.10 或更高版本 (instructions: for Windows)

  • OpenAI API key

可选

  • Memory backend (选择其中之一)

    • Pinecone

    • Milvus

    • Redis

    • Weaviate

  • ElevenLabs Key (如果你想AI说话)

⚠️ 配置 OpenAI API Keys ⚠️

从以下位置获取您的 OpenAI API 密钥: https://platform.openai.com/account/api-keys.

要将 OpenAI API 密钥用于自动 GPT，您需要设置结算信息（又名付费帐户）。

您可以在 https://platform.openai.com/account/billing/overview 设置付费帐户。

重要提示: 强烈建议您在 使用情况页面上跟踪使用情况。 您还可以在 使用情况限制页面上设置支出限制。

在继续之前，请确保您已完成此步骤。否则，什么都行不通！

💾 安装

要安装自动 GPT，请按照以下步骤操作：

  1. 确保您满足上面列出的所有环境要求。如果没有，请安装/获取它们。

若要执行以下命令，请导航到计算机上的文件夹并在顶部的文件夹路径中键入 CMD， 然后按 Enter 键，从而打开 CMD、Bash 或 Powershell 窗口。

  1. 克隆存储库：对于此步骤，您需要安装 Git。 注意：如果您没有 Git，只需下载 最新的稳定版本 或者在页面底部 (Source code (zip))。

    git clone -b stable https://github.com/Significant-Gravitas/Auto-GPT.git

  2. 进入下载存储库的目录。

    cd Auto-GPT

  3. 安装所需的依赖项。

    pip install -r requirements.txt

🔧 用法

  1. 在终端中运行 autogpt Python 模块。

    • 在 Linux/MacOS:

    ./run.sh

    • 在 Windows:

    .\run.bat

    在 .\run.bat 后面加上 --help 可以列出您可以传递的所有可能的命令行参数。

  2. 每次 Auto-GPT 响应后，从选项中选择授权命令，退出程序或向 AI 提供反馈。 i. 通过输入 y 授权单个命令。 ii. 通过输入 y -N 授权一系列连续的 N 命令。例如，输入 y -10 将运行 10 次自动迭代。 iii. 输入任何自由文本以向 Auto-GPT 提供反馈。 iv. 通过输入 n 退出程序。
