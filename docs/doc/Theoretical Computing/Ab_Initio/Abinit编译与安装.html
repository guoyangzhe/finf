Abinit 编译与安装

1. Abinit 简介

Abinit 是一个第一性原理计算软件，可以用于计算周期性体系的电子结构和光学性质。Abinit 的计算过程是基于密度泛函理论的，使用平面波基组和赝势，计算过程中使用了很多高效的算法，如快速傅里叶变换、快速多极子算法等。Abinit 的计算过程是自动化的，用户只需要提供一个输入文件，Abinit 就可以自动完成计算过程。

Abinit 的官方网站是：http://www.abinit.org/，在这里可以下载到 Abinit 的源代码和文档。Abinit 的源代码是开源的，可以在 GitHub 上找到：[

2. Abinit 的编译与安装

2.1 编译环境

Abinit 的编译需要一些依赖库，下面是 Abinit 官方网站上给出的编译环境：

  • C compiler: gcc, gfortran, icc, ifort, pgcc, pgf90, xlf, xlf90, …

  • MPI library: mpich, openmpi, …

  • BLAS library: atlas, mkl, …

  • LAPACK library: atlas, mkl, …

  • FFTW library: fftw2, fftw3, …

  • Libxc library: libxc, …

  • NetCDF library: netcdf, …

  • HDF5 library: hdf5, …

  • Libm library: libm, …

  • Libpthread library: libpthread, …

  • Librt library: librt, …

  • Libdl library: libdl, …

  • Libutil library: libutil, …

  • Libz library: libz, …

2.2 编译过程

Abinit 的编译过程比较简单，只需要执行下面的命令即可：

./configure --prefix=/path/to/abinit
make
make install
